Regular Expression Tutorial From the Wikipedia article on regular expressions:

A regular expression provides a concise and flexible means to "match" (specify and recognize)
strings of text, such as particular characters, words, or patterns of characters.

Consider the following sequence IDs in a FASTA file:
  • >Ken2100HCC
  • >Ken2101HCC
  • >Ken2215HCC
  • >Ken2220FIB
  • >Ken2226CHR
  • >Ken2228CHR
The following examples illustrate how to include or exclude various samples:

Regular ExpressionExplanation
IncludeHCCInclude only sequences with HCC in the sequence ID
ExcludeFIBExclude all sequences with FIB in the sequence ID
Exclude(2215|2220)Exclude Ken2215HCC and Ken2220FIB
Include(HCC|2228)Include only sequences with HCC or 2228 in the sequence ID
Include2..0Include only Ken2100HCC and Ken2220FIB
  • If the "regular expression" box is blank, this is interpreted to mean "all sequences".
  • The default setting is to set "Include" and to leave the regular expression box blank.
  • Specifying "Exclude" with a blankregular expression box will exclude all sequences.
